How to add photos

Close

Use these steps when you want to add a new album or update an existing one.

Step-by-step

  1. Enter an Album name (example: Youth Ministry, Choir, 2026 Revival).
  2. Add an optional Caption that will show under the photo.
  3. Add optional Tags (comma-separated), like youth, choir, or 2026.
  4. Select your photos and click Add Photos.
  5. (Optional) Use Arrange Album to change the order.
  6. Click Refresh Website Gallery so the public page gets the latest files.

Tips

  • Keep album names consistent to avoid duplicates (use Youth Ministry instead of both Youth and Youth Ministry).
  • Use tags for groups (ushers), events (revival), and years (2026).
Tip: If the gallery still looks stale, refresh the website gallery after your upload.
